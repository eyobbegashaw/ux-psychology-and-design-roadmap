# 🎨 UX Design & Behavioral Psychology Mastery

Welcome to my **UX Design Mastery** repository! This project is a deep dive into the intersection of user experience, human psychology, and product strategy. Following a structured behavioral design roadmap, this repo documents my journey in understanding how users think, decide, and interact with digital products.

## 📌 Project Overview
This repository serves as a professional design journal and portfolio. It covers the end-to-end UX process—from behavioral science and user research to wireframing and high-fidelity prototyping.

## 🧠 Core Focus Areas
* **Behavioral Design:** Applying Fogg’s Behavior Model and the Hook Model to drive user engagement.
* **Cognitive Psychology:** Reducing cognitive load and mastering human decision-making patterns[cite: 4].
* **Product Strategy:** Conducting SWOT analysis and building business model canvases for design success[cite: 4].
* **User Research:** Creating personas and mapping customer experience journeys[cite: 4].

## 🛠 Design Toolkit
* **Prototyping:** Figma, Adobe XD, Balsamiq[cite: 4].
* **Architecture:** BPMN, Flowcharts, and Product Backlogs[cite: 4].
* **Testing:** A/B Testing and Multivariate Testing[cite: 4].
